# Library Management System

## 📌 Description
This is a web-based Library Management System developed using PHP, HTML, CSS, and JavaScript.

## 🚀 Features
- User Registration & Login
- Admin Dashboard
- Book Management
- Borrow & Return Books

## 🛠️ Technologies Used
- PHP
- MySQL
- HTML
- CSS
- JavaScript

## ▶️ How to Run
1. Copy project folder to XAMPP htdocs
2. Start Apache & MySQL
3. Import database from /database folder
4. Open browser:
   http://localhost/LIBRARYMANAGEMENTSYSTEM/html/index.html
